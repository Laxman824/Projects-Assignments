- Cross check if you have task.cpp and Makefile in the same folder
- Open the terminal and type 'make' and enter
- After the compilation completes, type in ./task1 image_name.jpg (if your image is traffic.jpg, type ./task1 traffic.jpg)
- Your image should open up at this point, select four points of the road for analysis - Start from top-left and go clockwise
- A bird-eye's view of the road should open in a new window, hit Enter
- A croppped image of the road should open in a new window, hit Enter
- The program exits and both these images are saved in the same folder
